# 📚 Data Structures Basics Series  

## Topic 01: Linked Lists

Welcome to the **Data Structures Fundamentals Series** under  

**🚀 iAlgo – The Swift DSA Circle**

In this series, we will focus on **one data structure at a time** and understand it from the ground up. The goal is to build **strong fundamentals**, which are extremely important for solving DSA problems and cracking technical interviews at product-based companies.

Instead of jumping directly into difficult problems, we will first build clarity around **how data structures work internally**, how they are used, and how to implement them.

---

# 🎯 Objective of This Series

For each data structure, we will cover:

- Basic understanding and intuition  
- Internal structure and how it works  
- Traversal techniques  
- Basic CRUD operations (Create, Read, Update, Delete)  
- Fundamental algorithms related to it  
- Easy interview-style problems  

This approach helps in **building concepts step-by-step**, making it easier to understand advanced topics later.
