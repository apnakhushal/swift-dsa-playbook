# 📚 Patterns Covered So Far

## 1️⃣ Two Pointers Technique

The **Two Pointers** pattern is commonly used when working with **arrays or linked lists** where you need to iterate using two indices.

Instead of using nested loops, two pointers help **optimize time complexity**.

### Common Use Cases

- Pair problems
- Sorted array processing
- Removing duplicates
- Finding target sums

### Example Problems

- Two Sum (sorted array)
- Remove Duplicates from Sorted Array
- Container With Most Water
- Valid Palindrome

---

## 2️⃣ Sliding Window Pattern

The **Sliding Window** technique is used when dealing with **subarrays or substrings**.

Instead of recalculating values repeatedly, the window **slides across the data structure**, updating results dynamically.

This greatly improves performance for many problems.

### Common Use Cases

- Longest / shortest substring
- Maximum or minimum subarray
- Fixed or variable window problems

### Example Problems

- Longest Substring Without Repeating Characters
- Maximum Sum Subarray
- Minimum Window Substring
- Longest Repeating Character Replacement
